    • Button
    Buttons are very cruical part of any web applications. It allows user to interact with the application like submitting form, sending text, etc.
    Normal Button
    These buttons are used to grab user's attention as they are filled with specific colors. To use this, put class  btn  and variant name. 
                        
    /* Using the 'button' class on button along with 'variant' */ 

                <button class="btn">default</button>
                <button class="btn primary">primary</button>
                <button class="btn secondary">secondary</button>
                <button class="btn error">error</button>
                <button class="btn success">success</button>
    Outline Button
    Outline buttons serves the purpose when you want to add a functionality and don't want to grab immediate user's attention. To use this, include outline along with outline-variant within button tag. 
                        
    /* Using the 'outline-variant' class on button. */ 

        <button class="btn outline">default</button>
        <button class="btn outline-primary">primary</button>
        <button class="btn outline-secondary">secondary</button>
        <button class="btn outline-error">error</button>
        <button class="btn outline-success">success</button>
    Icon Button
    Icon buttons can be created using icons along with button text. Use class icon-btn. 
                        
    /* Using the 'icon-btn' class on button */ 

            <button class="btn icon-btn">
                <i class="your icon"></i>
                Save
            </button>
            <button class="btn icon-btn">
                <i class="your icon"></i>
                Reply
            </button>
    Link Button
    Link buttons can be created using class btn-link on a tag. 
                        
    /* Using the 'btn-link' class on 'a' tag */ 

            <button class="btn icon-btn">
                <a class="btn-link">
                    <i class="your icon"></i>
                    Link
                </a>
            </button>
    Disabled Button
    In order to use disabled buttons, include disabled attribute within button tag. 
                        
            <button class="btn primary">Disabled</button>
    Floating Action Button
    Floating action buttons provides a better user experience by allowing user to acess most used feature. Use floating-btn class within button tag. 
                        
            <button class="btn floating-btn primary">
                <i class="your icon"></i>
            </button>
