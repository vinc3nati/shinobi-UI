    • Avatar
    Avatars are used to display user profile pictures, usually found in circular shapes on navigation bars, blogs, social media profiles, etc.
    Normal Avatars for profiles
    For Normal Avatars use the class  avatar , inside the  img  tag. 
                        
                        /* Using the 'avatar' class name */ 

                <img class="avatar" src="url-of-image" alt="avatar_1">
                <img class="avatar" src="" alt="avatar_2">
    Sized Avatars
    Different sizes of avatars are  sm | md | lg  tag. 
                        
                        /* Using the 'sm | md | lg' class name */ 

                <img class="avatar sm" src="url-of-image" alt="avatar_1">
                <img class="avatar md" src="url-of-image" alt="avatar_2">
                <img class="avatar lg" src="url-of-image" alt="avatar_3">
    Squared Avatars
    Add a class  square  to the tag will acheive the square avatar.
    Note: 'square' class does not support svgs yet.
                        
                        /* Using the 'square' class name */ 

            <img class="avatar square sm" src="url-of-image" alt="avatar_1">
            <img class="avatar square md" src="url-of-image" alt="avatar_2">
            <img class="avatar square lg" src="url-of-image" alt="avatar_3">
    Text Avatar
    Text Avatars are used when profile pictures are not available. To use them, include  avatar-text  tag.
    SM
    MD
    LG 
                        
                        /* Using the 'avatar-text' class name */ 

            <div class="avatar avatar-text sm" src="url" alt="avatar_1"></div>
            <div class="avatar avatar-text md" src="url" alt="avatar_2"></div>
            <div class="avatar avatar-text lg" src="url" alt="avatar_3"></div>
