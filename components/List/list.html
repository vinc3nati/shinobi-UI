    • List
    List are the best way to show items specific to a certain category, eg: Gender, list of filters , etc.
    Normal List
    These are normal list where all the items are arranged one below another. Use class  list  on the container and  list-title  class on the title of the list.
    Company
  • Products
  • About
  • Testimonials
  • Contact
    •                     
        /* Using class 'list' on container and 'list-title' on heading. */

            <div class="list">
                <div class="list-title text-bold">Company</div>
                <li>Products</li>
                <li>About</li>
                <li>Testimonials</li>
                <li>Contact</li>
            </div>
    Nested List
    Nested lists are very handy to show nested data i.e. list of list. Use class  list  on the container and  nested-list  on the nested list container.
    Company
  • Products
  • About
    • News
    • Social Media
  • Testimonials
  • Contact
    •                     
        /* Using class 'list' on container and 'list-title' on heading. */

            <div class="list">
                <div class="list-title text-bold">Company</div>
                <li>Products</li>
                <li class="nested-list">
                    <div> About </div>
                    <ul>
                        <li>News</li>
                        <li>Social Media</li>
                    </ul>
                </li>
                <li>Testimonials</li>
                <li>Contact</li>
            </div>
