    Grids help us laying out a structure of an application and helps in responsive design.
    2 Column Grid
    This creates a grid with2 columns with equal width. To use it, include  layout-2-column  in your grid container. 
                        
    /* Using the 'layout-2-column' class on 'grid' container */ 

            <div class="grid layout-2-column">
                <div>
                    <img class="img img-responsive" src="url" alt="">
                </div>
                <div>
                    <img class="img img-responsive" src="url" alt="">
                </div>
            </div>
