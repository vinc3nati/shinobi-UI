    • Images
    Images are an integral part of an application as it conveys more content than text, and it's also improves user interface as well as user experience.
    Normal Images
    These are by default square images, with a default size of 150px vs 150px. Use class  img  in img tag. 
                        
            /* Using the 'img' class */ 

            <img class="img" src="url" alt="">
            <img class="img" src="url" alt="">
    Circle Images
    Circular Images are also have same dimensions i.e. 150px x 150px. Use class  img-round  along with img class. 
                                
            /* Using the 'img-round' class along with 'img' */ 
        
            <img class="img img-round" src="url" alt="">
            <img class="img img-round" src="url" alt="">
    Responsive Images
    Responsive Images are images which takes the dimensions of the container which is it in. Use class  img-responsive  along with img class. 
                        
            /* Using the 'img-round' class along with 'img' */ 
        
            <img class="img img-responsive" src="url" alt="">
