    • Alert
    Alerts are used to draw attention of the user to convey some important message, say notification, acknowledgement, error, etc
    Normal Alerts
    Normal Alerts can be used by including the class  alert  along with the variant name
    Default
    Primary
    Secondary
    Danger
    Warning
    Success
    Info 
                        
        /* Using the 'alert' class name along with 'variant' name */ 

                <div class="alert primary">Primary</div>
                <div class="alert secondary">Secondary</div>
                <div class="alert error">Error</div>
                <div class="alert danger">Danger</div>
                <div class="alert warning">Warning</div>
                <div class="alert success">Success</div>
                <div class="alert info">Info</div>
    Outlined Alerts
    Outlined Alerts can be used by including the class  alert  along with the outline-variant name.
    Primary Outline
    Secondary Outline
    Danger Outline
    Warning Outline
    Success Outline
    Info Outline 
                        
        /* Using the 'alert' class name along with 'outline-variant' name */ 

            <div class="alert outline-primary">Primary Outline</div>
            <div class="alert outline-secondary">Secondary Outline</div>
            <div class="alert outline-error">Error Outline</div>
            <div class="alert outline-danger">Danger Outline</div>
            <div class="alert outline-warning">Warning Outline</div>
            <div class="alert outline-success">Succes Outlines</div>
            <div class="alert outline-info">Info Outline</div>
