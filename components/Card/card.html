    • Card
    Cards are a great way to display data specific to a certain thing at one place. Cards are easier to look out and catches user's eye easily.
    Normal Card
    Normal cards contains a title , a description and a close icon. To use this, enclose  card-title  and  card-content  in a card container.
    Kakashi Hatake
    The master Copy Cat Ninja, with Chidori! Feared among the other villages as Kakashi of the Sharinghan. 
        
    /* Using the 'card-title' and 'card-content' class inside 'card' class */ 

            <div class="card">
                <div class="card-title">
                    Your title
                    <i class="fas fa-times card-close"></i>
                </div>
                <div class="card-content">
                    Your content
                </div>
            </div>
    Image Card
    Image cards contains a title,image, a description section and a close icon. To use this, enclose  card-image ,  card-title  and  card-content  in a card container.
    Kakashi Hatake
    The master Copy Cat Ninja, with Chidori! Feared among the other villages as Kakashi of the Sharinghan. 
        
    /* Using the 'card-image', 'card-title' and 'card-content' class 
        inside 'card' class */ 

            <div class="card">
            <img class="card-image" src="url" alt="">
                <div class="card-title">
                    Your title
                    <i class="fas fa-times card-close"></i>
                </div>
                <div class="card-content">
                    Your content
                </div>
            </div>
    Social Media Card
    Social Media cards contains a Avatar, Profile Name, Image title , a description and a close icon.
    Kakashi Hatake
    The master Copy Cat Ninja, with Chidori! Feared among the other villages as Kakashi of the Sharinghan. 
        
    /* Using classes 'card-header', 'card-title', 'card-image' and 
    'card-content' class inside 'card' class */ 

            <div class="card">
                <div class="card-header">
                    <div class="badge">
                        <img class="avatar" src="url" alt="user">
                        <div class="online"></div>
                    </div>
                    Username
                </div>
                <img class="card-image" src="url" alt="">
                <div class="card-content">
                    Your content
                </div>
            </div>
